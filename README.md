# Project Titan — Treino

App de treino (12 semanas · Push/Pull/Legs/Upper) com registro de carga por série,
gráfico de evolução, troca de exercício por músculo e fotos das máquinas Hammer Strength.

## Como usar
- **App:** abra o `index.html` (é autossuficiente — funciona offline, num único arquivo).
- **Estratégia:** `Estrategia_Titan_12semanas.pdf` com a periodização, cardio e nutrição de apoio.

## Publicar (GitHub Pages)
1. Suba estes arquivos para um repositório **público**.
2. Settings → Pages → Source: *Deploy from a branch* → Branch: `main` / `root` → Save.
3. Em ~1 min o app fica em `https://SEU-USUARIO.github.io/NOME-DO-REPO/`.
4. No Safari do iPhone: abrir o link → Compartilhar → **Adicionar à Tela de Início**.

## Arquivos
- `index.html` — versão atual do app (a que deve ser hospedada).
- `apple-touch-icon.png` — ícone da tela de início.
- `Estrategia_Titan_12semanas.pdf` — documento da estratégia.
- `project_titan_app_v*.html` — versões anteriores (histórico).
